Exercise 1: User Registration Form

Form Đăng Ký

Giải Thích CSS:

* selector - Áp dụng cho tất cả phần tử (reset margin, padding)
body - Style cho toàn trang (font, background, padding)
.container - Khung chứa nội dung (max-width, margin auto để căn giữa)
form - Style cho form (background, padding, border)
.form-group - Nhóm label + input (margin-bottom để tạo khoảng cách)
input - Style cho ô nhập (width 100%, padding, border)
input:focus - Hiệu ứng khi click vào input (đổi màu border)
button:hover - Hiệu ứng khi rê chuột lên button
transition - Hiệu ứng chuyển đổi mượt mà

Giải Thích Code JavaScript:

1. Lấy giá trị password từ input
2. Kiểm tra độ dài và có số/chữ không
3. Dựa vào kết quả để đánh giá Weak/Medium/Strong
4. Thêm class tương ứng để CSS tự động style

Kiến Thức CSS Quan Trọng:

Thuộc tính CSS Mô tả Ví dụ
padding Khoảng cách bên trong padding: 20px;
margin Khoảng cách bên ngoài margin: 10px;
border Viền border: 1px solid #ddd;
border-radius Bo góc border-radius: 8px;
background-color Màu nền background-color: #f5f5f5;
color Màu chữ color: #333;
font-family Font chữ font-family: Arial;
transition Hiệu ứng chuyển đổi transition: all 0.3s;
box-shadow Đổ bóng box-shadow: 0 2px 5px rgba(0,0,0,0.1);

CSS Selectors:

  • * - Tất cả phần tử
  • element - Theo tên thẻ (body, h1, p)
  • .class - Theo class (.container, .form-group)
  • #id - Theo ID (#username, #password)
  • element:hover - Khi rê chuột
  • element:focus - Khi click vào (input)
